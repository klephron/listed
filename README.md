# listed

List files with respect to ignore file.

## Installation

### Git

```sh
git clone git@github.com:klephron/listed.git
cd listed

pip install build
python -m build
pip install dist/listed-*.whl
```

### Sources

```sh
tar -xvf listed-*.tar.gz

pip install build
python -m build
pip install dist/listed-*.whl
```
